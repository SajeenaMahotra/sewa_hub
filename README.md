**SewaHub**
SewaHub is a cross-platform mobile application built with Flutter and Dart that connects homeowners with trusted local service providers for home maintenance tasks such as plumbing, electrical repairs, and cleaning. The app features service browsing, provider profiles, a real-time booking system, live messaging via Socket.IO, and an interactive map. It includes a unique emergency booking feature triggered by shaking the device. Built using Clean Architecture and MVVM pattern, with MongoDB Atlas for cloud storage and Riverpod for state management, SewaHub delivers a reliable, scalable, and offline-capable service platform.
